# Hexa Savunma

Hexa Teknoloji & Savunma kurumsal web sitesi — mühendislik, hassas imalat ve savunma sanayi çözümleri.

## Geliştirme

```bash
npm install
npm run dev
```

Tarayıcıda [http://localhost:3000](http://localhost:3000) adresini açın.

Önbellek sorunlarında:

```bash
npm run dev:clean
```
